Getting Started

Prerequisites

On Windows
  • Windows 10, 8.1, or 7
  • Git
  • Visual Studio 2015 Update 3 or newer
  • Option: CMake 3.12.4
On Linux or MacOS
  • gcc >= 7 or equivalent clang
  • Git
  • Option: CMake 3.12.4

1. Download vcpkg

> git clone https:github.com/Microsoft/vcpkg.git

> cd vcpkg

2. Build the Binary

On Windows

PS>   .\bootstrap-vspkg.bat

On Linux or MacOS

Linux:~/$   ./bootstrap-vcpkg.sh

3. Hook Up User-Wide Integration

On Windows

PS>   .\vcpkg integrate install

On Linux or MacOS

Linux:~/$   ./vcpkg integrate install

4. Install Any Package

On Windows

PS>   .\vcpkg install sdl2 curl

On Linux or MacOS

Linux:~/$   ./vcpkg install sdl2 curl

5. Use the Libraries with CMAKE

The best way to use installed libraries with CMake is via the toolchain file

scripts\buildsystems\vcpkg.cmake

To use this file, you simply need to add it onto your CMake command line as

-DCMAKE_TOOLCHAIN_FILE=[vcpkg root]\scripts\buildsystems\vcpkg.cmake

In Visual Studio, you can create a New Project (or open an existing one). All installed libraries are immediately ready to be #include'd and used in your project without additional configuration.
